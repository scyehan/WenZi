# Dictionary

EN↔ZH dictionary lookup via Youdao API with rich preview in the WenZi launcher.

## Features

### Search Source

Open the launcher and type `d ` followed by a word to look up:

| Source | Prefix | Description |
|--------|--------|-------------|
| `dictionary` | `d` | Youdao Dictionary (EN↔ZH) — auto-detects language direction |

- **English input** → Chinese definitions (en2zh-CHS)
- **Chinese input** → English definitions (zh2en)

### Preview Panel

When browsing results, a rich preview panel shows:
- Word header with US/UK phonetics (with audio buttons)
- Exam tags (CET4, IELTS, GRE, etc.)
- Parts of speech with translations
- Word forms (past tense, plural, etc.)
- Common phrases (up to 10)
- Synonyms grouped by part of speech
- Example sentences with translations (up to 5)
- Collins dictionary entries
- Etymology (Chinese)
- Web translations (ZH→EN fallback)

### Floating Window

Press Enter on a result to open the full definition in a standalone floating window (500×600). The window supports macOS dark mode.

## Usage

1. Open the WenZi launcher
2. Type `d hello` (or `d 你好` for Chinese)
3. Browse suggestions in the result list
4. View the rich preview panel on the right
5. Press Enter to open in a floating window

## Requirements

- WenZi ≥ 0.1.14
- Internet connection (Youdao API)
