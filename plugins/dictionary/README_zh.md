# Dictionary

通过有道 API 在 WenZi 启动器中进行英汉/汉英词典查询，支持富文本预览。

## 功能

### 搜索源

打开启动器，输入 `d ` 加上要查询的单词：

| 源 | 前缀 | 说明 |
|----|------|------|
| `dictionary` | `d` | 有道词典（英↔中）— 自动检测语言方向 |

- **英文输入** → 中文释义（en2zh-CHS）
- **中文输入** → 英文释义（zh2en）

### 预览面板

浏览结果时，右侧预览面板显示：
- 单词标题，附美式/英式音标（含发音按钮）
- 考试标签（四级、雅思、GRE 等）
- 词性及翻译释义
- 词形变化（过去式、复数等）
- 常用短语（最多 10 条）
- 近义词（按词性分组）
- 例句及翻译（最多 5 条）
- 柯林斯词典释义
- 词源（中文）
- 网络翻译（中→英备用）

### 浮动窗口

在结果上按 Enter 可打开独立浮动窗口（500×600）查看完整释义。窗口支持 macOS 深色模式。

## 使用方法

1. 打开 WenZi 启动器
2. 输入 `d hello`（或 `d 你好` 查询中文）
3. 在结果列表中浏览建议词条
4. 右侧预览面板查看详细释义
5. 按 Enter 在浮动窗口中打开完整内容

## 要求

- WenZi ≥ 0.1.14
- 需要网络连接（有道 API）
