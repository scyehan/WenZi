# Async Demo

WenZi 异步脚本功能的演示与验证插件。

## 功能

本插件注册了 **7 个启动器命令** 和 **1 个搜索源**，展示 WenZi 的异步脚本能力。

### 启动器命令

打开启动器，输入命令名称即可运行：

| 命令 | 说明 |
|------|------|
| `async-sleep` | 异步休眠，实时显示进度。可选参数：秒数（默认 3.0） |
| `async-fetch` | 异步抓取 URL 内容，保存到临时文件并打开。可选参数：URL 地址 |
| `async-timer` | 启动一个每 2 秒触发一次的异步定时器。按 `Ctrl+Cmd+T` 停止 |
| `async-concurrent` | 并行运行 3 个异步任务，报告总耗时 |
| `async-error` | 抛出 `RuntimeError` 异常 — 可在日志查看器中查看错误输出 |
| `async-run` | 通过 `wz.run()` 提交一个后台协程 |
| `async-pick` | 弹出选择器，异步处理用户选择 |

### 事件监听

| 事件 | 说明 |
|------|------|
| `transcription_done` | 语音转录完成时触发，记录日志并发送通知 |

### 搜索源

| 源 | 前缀 | 说明 |
|----|------|------|
| `async-search` | `as` | 模拟网络搜索（0.5 秒延迟），返回模糊匹配结果 |

## 使用方法

1. 打开 WenZi 启动器
2. 输入上述任意命令名称（如 `async-sleep`）
3. 按回车执行
4. 搜索源用法：输入 `as ` 加上搜索关键词

## 要求

- WenZi ≥ 0.1.12
- 无需额外依赖
