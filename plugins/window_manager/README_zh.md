# Window Manager

macOS 窗口管理插件，支持 Vim 风格的窗口吸附、居中和多显示器移动。

## 功能

使用熟悉的 Vim 风格按键（H/J/K/L）通过键盘快捷键管理窗口。

### 快捷键

| 快捷键 | 操作 | 说明 |
|--------|------|------|
| `Ctrl+Cmd+H` | 左半屏 | 将窗口吸附到屏幕左半部分 |
| `Ctrl+Cmd+L` | 右半屏 | 将窗口吸附到屏幕右半部分 |
| `Ctrl+Cmd+K` | 上半屏 | 将窗口吸附到屏幕上半部分 |
| `Ctrl+Cmd+J` | 下半屏 | 将窗口吸附到屏幕下半部分 |
| `Ctrl+Cmd+R` | 全屏 | 最大化窗口至整个屏幕 |
| `Ctrl+Cmd+W` | 居中 | 将窗口居中显示 |
| `Ctrl+Cmd+E` | 下一屏 | 将窗口移动到下一个显示器 |

## 使用方法

插件安装并启用后，所有快捷键立即生效。聚焦任意窗口后按对应快捷键即可。

### 多显示器

使用 `Ctrl+Cmd+E` 可在多个显示器之间循环移动窗口。配合吸附快捷键，可在多屏间精确定位窗口。
