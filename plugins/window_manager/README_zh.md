# Window Manager

macOS 窗口管理插件，支持 Vim 风格的窗口吸附、居中和多显示器移动。

## 功能

使用熟悉的 Vim 风格按键（H/J/K/L）通过键盘快捷键管理窗口。

### 快捷键

| 快捷键 | 操作 | 说明 |
|--------|------|------|
| `Ctrl+Cmd+H` | 左吸附 | 吸附到屏幕左侧，循环切换 1/2 → 2/3 → 1/3 宽度 |
| `Ctrl+Cmd+L` | 右吸附 | 吸附到屏幕右侧，循环切换 1/2 → 2/3 → 1/3 宽度 |
| `Ctrl+Cmd+K` | 上吸附 | 吸附到屏幕上方，循环切换 1/2 → 2/3 → 1/3 高度 |
| `Ctrl+Cmd+J` | 下吸附 | 吸附到屏幕下方，循环切换 1/2 → 2/3 → 1/3 高度 |
| `Ctrl+Cmd+R` | 全屏 | 最大化窗口至整个屏幕 |
| `Ctrl+Cmd+W` | 居中 | 将窗口居中显示 |
| `Ctrl+Cmd+E` | 下一屏 | 将窗口移动到下一个显示器 |

### 循环吸附

重复按同一方向键会循环切换三种尺寸：**1/2 → 2/3 → 1/3** 屏幕大小（类似 Hammerspoon 的 miro-windows-manager）。按不同方向键则重置为 1/2。

## 使用方法

插件安装并启用后，所有快捷键立即生效。聚焦任意窗口后按对应快捷键即可。

### 多显示器

使用 `Ctrl+Cmd+E` 可在多个显示器之间循环移动窗口。配合吸附快捷键，可在多屏间精确定位窗口。

## 要求
