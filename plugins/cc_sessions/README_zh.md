# Claude Code Sessions

通过 WenZi 启动器浏览和查看 Claude Code 会话历史。

## 功能

### 会话搜索

打开启动器，输入 `cc ` 即可搜索 Claude Code 会话。

**搜索能力：**
- 按会话标题、项目名、Git 分支或消息数进行模糊搜索
- 项目过滤：输入 `@项目名` 加搜索词（如 `@WenZi window`）
- 可搜索会话摘要和首条用户消息

**每条结果显示：**
- 会话标题（自定义标题、摘要或首条提示）
- 项目名、相对时间（如"2 小时前"）、Git 分支
- 消息数量角标
- 自动生成的项目头像

### 操作

| 按键 | 操作 |
|------|------|
| `Enter` | 在详细查看器中打开会话 |
| `Cmd+Enter` | 复制会话文件路径到剪贴板 |
| `Delete` | 将会话移至废纸篓（需确认） |

### 预览面板

浏览结果时，预览面板显示：
- 会话标题和元数据标签（项目、分支、Claude 版本、消息数/Token 数）
- 时间信息（创建、修改、持续时长）
- 最近 10 轮对话的文本预览

### 会话查看器

在搜索结果上按 Enter 打开完整查看器：
- 完整对话渲染，支持 Markdown 格式和代码语法高亮
- 可展开/收起的工具调用和思考过程
- 代码块一键复制
- **实时自动刷新** — 活跃会话进行中，查看器自动更新最新内容
- 子代理支持 — 自动检测并链接到衍生的子代理会话

### 缓存管理

在启动器中执行 `cc-sessions:clear-cache` 命令可重置缓存的会话元数据，强制重新扫描。

## 使用方法

1. 打开 WenZi 启动器
2. 输入 `cc ` 加上搜索关键词
3. 浏览结果，按 Enter 查看会话
4. 使用 `@项目名` 语法按项目过滤

## 要求

- WenZi ≥ 0.1.12
- Claude Code 会话存储在 `~/.claude/projects/`
